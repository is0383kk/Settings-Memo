# CLAUDE.md

## 注意事項

### 環境について

Claude Code は Windows11 上の PowerShell で実行されています

### 使用言語

- 日本語を使う
- 文字化けに注意すること
- 文字コードは UTF-8 を使う

### 影響範囲の確認

- 意図しない変更や影響が発生する箇所を特定して作業すること
- 許可なくサーバの起動を行わないこと（npm run dev など）

### 曖昧さの回避

- 「～の可能性があります」「～かもしれません」を禁句とし、必ず一次情報を根拠に作業すること
- 根拠となるソースの URL と該当箇所の引用を必ずユーザに知らせること

### 指示にないスタイリングの追加

- 独断で UI のデザインを変更しないこと

### エラーの隠蔽

- エラーの隠蔽をしないこと
- 失敗しているテストをスキップしない
