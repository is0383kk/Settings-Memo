# はじめに

本手順書では WSL2 での各種操作方法について説明する。

## ■ VSCode で WSL プロジェクトを開く

### WSL のコマンドライン上から VSCode を開く

WSL ディストリビューションからプロジェクトを開くには、ディストリビューションのコマンドラインを開き、下記入力する。

```
$ code .
```

## WSLのディストリビューションを削除する

現在のディストリビューションを確認する

```powershell
$ wsl --list --verbose
  NAME      STATE           VERSION
* Ubuntu    Stopped         2
```

対象のディストリビューションを削除する

```
$ wsl --unregister Ubuntu
登録解除。
この操作を正しく終了しました。
```

再度、ディストリビューションを確認する

```
$ wsl --list --verbose
Linux 用 Windows サブシステムにインストールされているディストリビューションはありません。
この問題を解決するには、以下の手順に従ってディストリビューションをインストールしてください:
```
